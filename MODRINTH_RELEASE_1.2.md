## Race Timers 1.2

Client-side race timer HUD for Minecraft Fabric.

Supports elytra, boats, horses, and other route-based runs.

### Highlights

- Lap-based timing with per-world track storage
- Shared HUD settings across all worlds
- Mod Menu config screen powered by AlinLib
- Commands under `/lap`, with legacy `/circle` alias

### Supported Versions

- Minecraft `1.21` to `1.21.11`
- Fabric Loader `0.18.2+`

### Main Commands

- `/lap help`
- `/lap list`
- `/lap set <id>`
- `/lap target <count>`
- `/lap run reset`
- `/lap run stop`
